# User Guide

## Browsing Doctors

- Scroll down to load more doctors automatically.
- Each card shows name, specialty, rating, and call types.

## Booking an Appointment

- Click "Book Appointment" on a doctor card.
- Select a date and time in the modal.
- Click "Confirm Booking" to finish.
- You’ll see a confirmation and can close the modal.

## Troubleshooting

- If doctors don’t load, check your internet connection or try refreshing.
- If images are missing, a default doctor image will be shown.

## FAQ

**Q: How do I change the theme?**
A: The app follows your system/browser dark/light mode automatically.

**Q: Why do I see skeleton cards?**
A: Skeletons appear while new doctors are loading for a smoother experience.

**Q: How do I cancel a booking?**
A: Click "Cancel" in the booking modal before confirming.
